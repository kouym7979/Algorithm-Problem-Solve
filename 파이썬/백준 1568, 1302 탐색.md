## 백준 1568, 1302 탐색

1568번

```
n = int(input())

time = 0
cnt = 1

while True:
    if n == 0:
        break

    if n >= cnt:
        time += 1
        n -= cnt
        cnt += 1

    if 0 < n < cnt:
        cnt = 1

print(time)
```

1302번

```
# 등장횟수를 구할 경우 dictionary를 이용하는게 좋다
n = int(input())

book = {}
cnt = []
for i in range(n):
    title = input()
    if title not in book:
        book[title] = 1
    else:
        book[title] += 1

target = max(book.values())
result = []

for title, num in book.items():
    if num == target:
        result.append(title)

result.sort()
print(result[0])
```
